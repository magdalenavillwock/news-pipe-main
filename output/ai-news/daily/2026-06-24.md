### Anthropic & Claude

#### Claude Code Releases — [v2.1.187](https://github.com/anthropics/claude-code/releases/tag/v2.1.187)

**Neue Features:**
- **`sandbox.credentials`-Einstellung**: Sandboxed Commands können nun daran gehindert werden, Credential-Dateien und geheime Umgebungsvariablen zu lesen — wichtiges Sicherheitsfeature für isolierte Agenten-Umgebungen.
- **Org-weite Modellrestriktionen**: Administratoren können Modelle einschränken; der Model-Picker, `--model`, `/model` und `ANTHROPIC_MODEL` zeigen bei Verstoß eine Meldung „restricted by your organization's settings".
- **Mausklick-Support in Fullscreen-Menüs**: Klickbare Auswahlmenüs in Permission-Prompts, `/model`, `/config` etc. im Fullscreen-Modus.
- **Verbessertes `/install-github-app`**: GitHub Actions Workflow-Setup ist jetzt optional — man kann die GitHub App alleine installieren.
- **`/btw` mit Pfeiltasten-Navigation**: Links/Rechts durch frühere Antworten blättern.
- **`/plugin` zeigt ungenutzte Plugins**: Erleichtert die Bereinigung selten genutzter Plugins.

**Bugfixes:**
- `--resume` schlug fehl mit „No conversation found" wenn der ursprüngliche `-p`-Run keine Modellantworten produziert hatte.
- `--json-schema` und Workflow `agent({schema})`: Das Modell kann `StructuredOutput` nicht mehr unendlich wiederholt aufrufen; Folge-Turns liefern zuverlässig strukturierte Ausgaben.
- Remote-MCP-Tool-Calls, die 5 Minuten lang hingen, werden jetzt mit einem Fehler abgebrochen (überschreibbar mit `CLAUDE_CODE_MCP_TOOL_IDLE_TIMEOUT`).
- Claude Code Remote-Sessions starteten ~2,7 Sekunden langsamer nach dem CA-System-Trust-Install — behoben.
- Eingeklebter koreanischer/CJK-Text wurde in Terminals, die Paste als Byte-für-Byte Extended-Key-Events liefern, zu Mojibake.
- `/update` über Remote Control hing wenn ein Startup-Trust-Dialog erschienen wäre.
- Background-Jobs in der Agenten-Ansicht blieben dauerhaft auf „working", wenn der Agent ohne strukturierte Ausgabe endete.
- Channel-Verbindungen brachen nach Navigation zur Agenten-Ansicht und zurück, sowie nach `/bg`, `/tui` oder `/update` ab.
- Agenten-Stop-Benachrichtigungen zeigten falsch an, wer den Agenten gestoppt hat; Formulierung verbessert auf „finished"/„stopped".
- Subagenten-Tiefenverfolgung: Fortgesetzte Subagenten erhalten ihre ursprüngliche Spawn-Tiefe zurück; geforkte Subagenten zählen korrekt zur Tiefengrenze.
- Geleakte Worktree-Registrierungen: Gesperrte `.git/worktrees/`-Einträge von getöteten Agenten werden automatisch bereinigt.
- Cmd+Click öffnete URLs in Ghostty (macOS) im Fullscreen-Modus nicht.
- `claude --help` listete das `--bg`/`--background`-Flag nicht auf.
- Esc, Ctrl-C und Ctrl-D funktionierten während `/share` hochlud nicht.
- **[VSCode]** Extension wurde beim Fortsetzen einer großen Session unresponsiv.

#### Anthropic — Claude Tag (TechCrunch & VentureBeat)

Anthropic launcht **[Claude Tag](https://techcrunch.com/2026/06/23/anthropics-claude-tag-is-learning-your-company-one-slack-message-at-a-time/)** — eine tiefe Slack-Integration, die Claude als dauerhaften, kanalweiten KI-Teamkollegen einbettet. Die wichtigsten Eigenschaften:

- **Multiplayer**: Ein geteilter Claude pro Kanal — alle Teammitglieder interagieren mit derselben Instanz, können laufende Konversationen aufgreifen.
- **Gedächtnis**: Claude akkumuliert Kanalkontext über Zeit; kein erneutes Einführen von Projekten notwendig.
- **Proaktivität (Ambient Mode)**: Claude überwacht Kanäle und bringt sich von sich aus mit relevanten Informationen ein, ohne explizit gefragt zu werden.
- **Asynchrone Arbeit**: Claude arbeitet eigenständig über Stunden oder Tage an Aufgaben.
- **Sicherheit & Governance**: Admins definieren separate Claude-Identitäten pro Kanal mit eigenem Tool-Zugriff, Spending-Limits und vollständigem Aktionslog.

Läuft auf **Claude Opus 4.8**. Derzeit in Beta für Enterprise- und Team-Kunden. VentureBeat berichtet: 65 % von Anthropics eigenem Produkt-Code wird durch die interne Version von Claude Tag generiert. Anthropics Jahresumsatz liegt inzwischen bei ca. 47 Mrd. Dollar Run-Rate, Claude Code allein bei über 2,5 Mrd.

### 📄 ArXiv Papers

- **[InSight: Self-Guided Skill Acquisition via Steerable VLAs](http://arxiv.org/abs/2606.24884v1)** — Framework, das Vision-Language-Action-Modelle auf Primitive-Aktions-Ebene steuerbar macht (z.B. „move gripper to bowl"). Automatisierte Segmentierung von Demonstrations-Daten in labeled Primitives ermöglicht autonome Skill-Akquisition jenseits der Trainingsdaten.

- **[FLUX3D: High-Fidelity 3D Gaussian Generation](http://arxiv.org/abs/2606.24874v1)** — Neue Methode für Image-to-3DGS-Generierung mit diffusionsausgerichteter Sparse-Repräsentation. Adressiert zwei Engpässe: Diskriminative 2D-Features unterdrücken rekonstruktive Hinweise; Standard-Diffusion-Transformer fehlen strukturelles Bewusstsein.

- **[OpenThoughts-Agent: Data Recipes for Agentic Models](http://arxiv.org/abs/2606.24855v1)** — Vollständig offene Datenkurations-Pipeline für agentische Modelle. Bestehende Ansätze (SWE-Smith, SERA, Nemotron-Terminal) trainieren nur auf einzelnen Benchmarks — OT-Agent adressiert die Generalisierung über diverse agentische Aufgaben.

- **[SHERLOC: Structured Diagnostic Localization for Code Repair Agents](http://arxiv.org/abs/2606.24820v1)** — LLM-Agenten verbrauchen die Hälfte ihres Budgets für Fehlerlokalisation. SHERLOC kombiniert Reasoning-LLMs mit kompakter Repräsentation für strukturierte, hypothesengetriebene Diagnose ohne Training — liefert actionable Diagnose statt reines File-Retrieval.

- **[IV-CoT: Implicit Visual Chain-of-Thought for Text-to-Image Generation](http://arxiv.org/abs/2606.24849v1)** — Struktur-bewusste Text-zu-Bild-Generierung durch latenten visuellen Reasoning-Schritt. Trennt strukturelles Planen von visueller Darstellung für bessere Prompt-Treue bei Objektzählung, räumlichen Relationen und Attributbindungen.

- **[Matching Tasks to Objectives: Fine-Tuning für Encoder-Decoder Modelle](http://arxiv.org/abs/2606.24841v1)** — Studie zum Einfluss verschiedener Pre-Training-Objectives auf Encoder-Decoder-LLMs. Neues MTO (Match Task to Objective)-Framework für Common-Sense-Wissen und Generierungsaufgaben.

- **[Less is More: Quality-Aware Training Data Selection](http://arxiv.org/abs/2606.24828v1)** — Wissenschaftliche Zusammenfassungs-Datensätze verwenden Abstracts als Goldstandard, obwohl deren Qualität variiert. Paper veröffentlicht einen der größten biomedizinischen Langdokument-Datensätze mit qualitätsgesteuerter Auswahl.

- **[AI-Powered AAC Interfaces](http://arxiv.org/abs/2606.24854v1)** — Evaluation von KI-gestützten Augmentative and Alternative Communication (AAC) Systemen. Zeigt, warum gängige Metriken die vielschichtigen Nutzeranforderungen nicht vollständig erfassen.

- **[Paying to Know: Micro-Transaction Markets for Agentic E-Commerce](http://arxiv.org/abs/2606.24783v1)** — Mit autonomen Agenten und Mikrozahlungsschienen (x402, AP2) verlagert sich der E-Commerce-Engpass vom Produkt-Matching zur Vertrauenswürdigkeit von Produktinformationen. Agenten können exhaustiv recherchieren — das Neue ist der Markt für verifizierte Daten.
