**[Claude Code v2.1.146](https://github.com/anthropics/claude-code/releases/tag/v2.1.146)** — Veröffentlicht 21. Mai 2026

Vollständige Liste der Änderungen:

**Neue Features / Umbenennungen:**
- `/simplify` wurde in `/code-review` umbenannt, mit optionalem Effort-Level (z.B. `/code-review high`)
- Auto-Mode unterdrückt `AskUserQuestion` nicht mehr, wenn der User oder ein Skill explizit darauf angewiesen ist

**Bugfixes:**
- Windows PowerShell-Tool schlug mit "command line is invalid" fehl, wenn `pwsh` via Winget oder Microsoft Store installiert war (Regression seit v2.1.124) — behoben
- MCP `resources/list`, `resources/templates/list` und `prompts/list` verloren Items ab Seite 2 bei paginierenden Servern — behoben
- Vollbild-Strobing in angehängten Background-Sessions unter Windows Terminal beim Streamen — behoben
- Auto-Updater-Statusleiste zeigte beim Update-Fehler nicht die aktuelle Version an — behoben
- Auf Windows: Entfernen eines Background-Job-Worktrees folgte nicht mehr NTFS-Junctions in das Haupt-Repo — behoben
- `/background` lehnte Sessions ab, deren einziger Input ein Skill oder Custom Slash Command war — behoben
- Backgroundete Sessions fragten erneut nach Tool-Permissions, die bereits mit "don't ask again" erteilt wurden — behoben
- `/theme` Color Editor und "New custom theme"-Dialog reagierten nicht auf Esc — behoben
- Uncaught Exception am Ende von Streaming-Sessions im Agent SDK — behoben
- `forceLoginOrgUUID` und `forceLoginMethod` Managed-Settings-Policies wurden bei Third-Party-Provider- und API-Key-Sessions nicht durchgesetzt — behoben
- GNOME Terminal: Rechtsklick- und Mittelklick-Paste fügte keinen Text ein — behoben
- `CLAUDE_CODE_SUBAGENT_MODEL` wurde in Multi-Agent-Sessions nicht an Kindprozesse weitergegeben — behoben

**Verbesserungen:**
- Auto-Updater zuverlässiger: Native Versionsprüfungen und Downloads wiederholen bei transienten Netzwerkfehlern statt sofort abzubrechen
- Diff-Rendering-Performance für große Datei-Edits verbessert
