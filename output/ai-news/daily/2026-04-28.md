### Anthropic & Claude

Die wichtigste Entwicklung des Tages kommt von Anthropic mit dem umfangreichen [Claude Code v2.1.121 Release](https://github.com/anthropics/claude-code/releases/tag/v2.1.121):

**Neue Features:**
- `alwaysLoad`-Option für MCP Server Config umgeht Tool-Search-Deferral komplett  
- `claude plugin prune` entfernt verwaiste Auto-Install-Dependencies
- Type-to-Filter Suchbox in `/skills` für bessere Navigation
- PostToolUse Hooks können nun Tool Output für alle Tools via `hookSpecificOutput.updatedToolOutput` ersetzen
- Fullscreen Mode: Tippen in Prompt springt nicht mehr automatisch nach unten
- Scrollbare Dialoge mit Pfeil-, PgUp/PgDn-Tasten und Mausrad
- Klicken auf umgebrochene URLs öffnet vollständige URL
- SDK: `CLAUDE_CODE_FORK_SUBAGENT=1` funktioniert in non-interaktiven Sessions

**Performance & Fixes:**
- Memory-Leak behoben: Unbegrenztes Speicherwachstum bei vielen Bildern (mehrere GB RAM)
- `/usage` Memory-Leak gefixt (bis zu 2GB auf Maschinen mit großen Transkript-Historien)
- Bash Tool wird nicht mehr permanent unbrauchbar wenn Start-Verzeichnis gelöscht wird
- `--resume` Crashes behoben, korrupte Transkript-Zeilen werden übersprungen
- Microsoft 365 MCP OAuth Fehler mit doppelten Parametern behoben

**Zusätzliche Verbesserungen:**
- MCP Server Auto-Retry bis zu 3x bei Startup-Fehlern
- Terminal Session Title in konfigurierter Sprache generiert
- Claude.ai Connector-Deduplizierung
- Vertex AI: X.509 Certificate-based Workload Identity Federation Support
- OpenTelemetry: `stop_reason`, `gen_ai.response.finish_reasons` zu LLM Request Spans hinzugefügt

Das [Claude Agent SDK v0.2.121](https://github.com/anthropics/claude-agent-sdk-typescript/releases/tag/v0.2.121) bringt ebenfalls Neuerungen:
- `updatedToolOutput` zu `PostToolUseHookSpecificOutput` für alle Tools hinzugefügt
- `updatedMCPToolOutput` wird als deprecated markiert

Zusätzlich gab es ein [Update zu Claude Code Quality Reports](https://www.anthropic.com/engineering/april-23-postmortem), das sich auf aktuelle Qualitätsprobleme bezieht.

### OpenAI & Microsoft

Die größte Branchennews ist die [Neustrukturierung der Microsoft-OpenAI Partnerschaft](https://openai.com/index/next-phase-of-microsoft-partnership/). Die wichtigsten Änderungen:

**Beendete Exklusivität:**
- OpenAI kann alle Produkte auf beliebigen Cloud-Providern anbieten
- Microsoft zahlt keine Revenue-Share mehr für Azure-OpenAI-Zugriffe  
- AGI-Klausel komplett entfernt, ersetzt durch feste Zeiträume bis 2032
- Microsofts Lizenz wird von exklusiv zu non-exklusiv

**Finanzielle Neustrukturierung:**
- OpenAI zahlt weiter 20% Revenue-Share an Microsoft bis 2030, aber mit Cap
- Microsoft behält ~27% Ownership an OpenAI's For-Profit-Entity
- Ermöglicht OpenAIs $50B Amazon-Deal ohne rechtliche Konflikte

[Satya Nadella kündigte](https://www.linkedin.com/posts/satyanadella_agent-mode-is-here-in-outlook-copilot-can-activity-7454646422165012480-Q1Be) außerdem **Agent Mode in Outlook** an - Copilot kann nun Postfächer verwalten, E-Mails triagieren und Meetings umplanen.
