## 2. Trend-Analyse

### A) Die Agent-Ära wird konkret
Alle großen Player liefern diese Woche Agent-Infrastruktur: OpenAI mit Workspace Agents, AWS mit Bedrock AgentCore, Microsoft mit Hosted Agents in Foundry, Anthropic mit Managed Agents Scaling und einem experimentellen Agent-zu-Agent-Marketplace. Google TPU 8i ist explizit für "agentic inference" optimiert. Der Übergang von Chatbot zu autonomem Agent vollzieht sich in Produktionsumgebungen.

### B) Chinesische Konkurrenz verschärft den Preisdruck
DeepSeek V4 ($1.74/$3.48 pro 1M Tokens vs. $35 für GPT-5.5) demonstriert, dass frontier-nahe Performance zu Bruchteilen der Kosten möglich ist. Die Huawei-NPU-Kompatibilität signalisiert zudem wachsende Hardware-Unabhängigkeit von NVIDIA.

### C) Massive Kapitalkonzentration in wenigen Playern
Google $40B in Anthropic, SpaceX $60B für Cursor, Microsoft A$25B in Australien allein — die AI-Infrastrukturinvestitionen erreichen eine neue Größenordnung. Elizabeth Warrens Bubble-Warnung ist im Kontext dieser Zahlen ernst zu nehmen.

### D) Security wird zum Achillesferse der AI-Adoption
Vercel-Breach über kompromittiertes AI-Tool, Anthropic Mythos in falsche Hände geraten, Meta trackt Mitarbeiter-Keystrokes, Uni-Domains dienen Porn — die Angriffsfläche wächst schneller als die Abwehr. VentureBeat attestiert 72% der Unternehmen eine "Governance Mirage".

### E) Europa emanzipiert sich
Cohere+Aleph Alpha Fusion mit Regierungsunterstützung aus Deutschland und Kanada, Schwarz Group Backing — erstmals eine glaubwürdige europäische Alternative zu US-AI für Enterprise.

## 6. Anthropic & Claude — Detailübersicht

### Claude Code v2.1.117 (22. April)

**Neue Agent-Features:**
- Subagent-Forking für externe Builds aktivierbar via `CLAUDE_CODE_FORK_SUBAGENT=1`
- Agent-Frontmatter `mcpServers` werden für Hauptthread-Agent-Sessions geladen
- Verbesserte Parallelisierung bei unabhängigen Queries

**Benutzererfahrung:**
- `/model`-Befehl: Auswahl bleibt nach Neustarts bestehen, Header zeigt aktives Modell-Pinning
- `/resume`-Command: Bietet Zusammenfassung für veraltete, große Sessions vor erneutem Laden
- Schnellerer Startup bei gemischten lokalen/claude.ai MCP-Server-Configs

**Plugin-System:**
- `plugin install` installiert jetzt fehlende Dependencies statt "bereits installiert"-Stopp
- Dependency-Fehler zeigen "not installed" mit Install-Hinweis
- Auto-Auflösung fehlender Dependencies aus konfigurierten Marketplaces
- Durchsetzung von `blockedMarketplaces` und `strictKnownMarketplaces`

**Advisor Tool (Experimental):**
- Dialog trägt jetzt "experimental" Label mit Learn-More Link
- Startup-Benachrichtigung wenn aktiviert
- Behebt "Advisor tool result content could not be processed"-Fehler

**Performance:**
- `cleanupPeriodDays` deckt jetzt auch `~/.claude/tasks/`, `shell-snapshots/`, `backups/` ab
- Native Builds: Embedded `bfs` und `ugrep` für schnellere Suchen
- Windows: Cached `where.exe` Lookups für schnellere Subprocess-Starts
- Standard-Effort für Pro/Max auf Opus 4.6/Sonnet 4.6 ist jetzt "high" (war "medium")

**Bugfixes:**
- OAuth-Token-Refresh bei 401-Fehlern in Plain-CLI-Sessions
- WebFetch hängt nicht mehr bei sehr großen HTML-Seiten
- Proxy HTTP 204 No Content wird korrekt behandelt
- Prompt-Input Undo funktioniert jetzt korrekt
- NO_PROXY wird bei Remote-API-Requests unter Bun respektiert
- Opus 4.7 Context-Berechnung korrigiert (1M statt 200K Window)
