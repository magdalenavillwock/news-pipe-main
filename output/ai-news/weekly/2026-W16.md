## 2. Trend-Analyse

### Agent-Ära wird ernst
Die Woche markiert einen Wendepunkt: Sowohl Anthropic (Managed Agents Paper, Claude Code Auto Mode, 90+ Plugins) als auch OpenAI (Codex Computer Use) und Microsoft (AgentCore SDK GA) liefern produktionsreife Agent-Infrastruktur. Databricks zeigt: Multi-Step Agents schlagen Single-Turn RAG um 21%. Die Architektur-Frage "Brain vs. Hands" (Anthropics Paper) wird zum zentralen Design-Pattern.

### Coding-Tools: Vom Assistenten zum Autonomen
Claude Code mit Auto Mode, OpenAI Codex mit Computer Use, Factory mit $1,5B Bewertung — Coding-Agents werden autonom. Gleichzeitig warnt TechCrunch vor "Tokenmaxxing": Mehr generierter Code ≠ mehr Produktivität. 43% der AI-generierten Code-Changes brauchen Production-Debugging.

### Trust-Krise bei OpenAI
Farrow-Exposé, physische Angriffe auf Altman, Sora-Team-Exodus, interne Memos über Konkurrenz-Angst — OpenAI steht unter massivem Druck. Anthropics $380B Bewertung wirkt laut TechCrunch als "Schnäppchen" vs. OpenAIs $1,2T IPO-Annahme.

### Hardware-Bottleneck verschärft sich
RAM-Shortage bis 2030, Meta erhöht Quest-Preise wegen RAM-Mangel, NVIDIAs "Cost per Token" als neue TCO-Metrik. Die physische Infrastruktur wird zum limitierenden Faktor der AI-Revolution.

### Shadow AI & Security werden kritisch
82% der Unternehmen berichten AI-Agent-Security-Incidents (VentureBeat). Shadow AI 2.0 durch lokale LLMs auf Laptops. OpenAI startet Cyber Defense Ecosystem mit GPT-5.4-Cyber und $10M API Grants. Data Drift untergräbt bestehende ML-Sicherheitsmodelle.
